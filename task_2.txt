Практическое задание по теме “Управление БД”.

1.1 Выполняем команду в консоли -> sudo apt-get install mysql-server mysql-client mysql-common php7.0-mysql
1.2 Создаём в корне файл .my.cnf -> touch ./.my.cnf
1.3 Добавляем в пустой файл .my.cnf строки: 	[client]
						user=root
						password="пароль"
1.4 Запускаем в консоли mysql и убеждаемся что ввода пароля не требуется.

2.1 Создаём в mysql базу данных example -> CREATE DATABASE example;
2.2 В консоли пишем ->	USE example;
			DROP TABLE IF EXISTS users;
			CREATE TABLE users (
    			id SERIAL PRIMARY KEY,
    			name VARCHAR(255),
    			UNIQUE unique_name(name(10)));
2.3 Добавляем в таблицу пользователей -> INSERT INTO users (name) VALUES ('Vasya');
					 INSERT INTO users (name) VALUES ('Ivan');
2.4 Смотрим что получилось ->		 SELECT * FROM users;
					 +----+-------+
					 | id | name  |
					 +----+-------+
					 |  1 | Vasya |
					 |  2 | Ivan  |
					 +----+-------+

3.1 Выходим из mysql
3.2 В консоли выполняем команду -> mysqldump -u root -p example > backup.sql
3.3 Разворачиваем содержимое дампа в новую базу данных sample, выполнив -> mysql -u root -p --default-character-set=utf8 sample < backup.sql
3.4 Заходим в mysql, в базу данных sample -> USE sample;
3.5 Проверяем -> SELECT * FROM users;
		 +----+-------+
		 | id | name  |
		 +----+-------+
		 |  1 | Vasya |
		 |  2 | Ivan  |
		 +----+-------+

4.1 Для того чтобы создать дамп единственной таблицы help_keyword базы данных mysql и чтобы дамп содержал только первые 100 строк таблицы в 
консоли необходимо выполнить -> mysqldump -u root -p --where="true limit 100" mysql > backup_help_keyword.sql
